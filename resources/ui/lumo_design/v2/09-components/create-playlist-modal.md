# Create Playlist Modal

> 新建歌单弹窗，从 SidebarLeft "新建播放列表" 触发。

## Anatomy

```
┌─────────────────────────────────┐
│  新建歌单                        │
│  创建一个新的播放列表              │
│                                 │
│  名称                            │
│  ┌─────────────────────────────┐│
│  │ 输入歌单名称                 ││
│  └─────────────────────────────┘│
│  描述（可选）                    │
│  ┌─────────────────────────────┐│
│  │ 添加描述…                   ││
│  └─────────────────────────────┘│
│                                 │
│             [取消]    [创建]     │
└─────────────────────────────────┘
```

## States

| 状态 | 行为 |
|---|---|
| 默认 | 名称为空，创建按钮禁用 |
| 输入中 | 名称非空，创建按钮可用 |
| 提交中 | 按钮显示"创建中…"，不可重复点击 |
| 成功 | 关闭弹窗，自动切换到新歌单 |
| 失败 | 显示错误信息"创建失败，请重试" |

## 交互细节

- 点击遮罩层（`bg-black/30`）关闭弹窗
- 回车键（Enter）= 确认提交
- 创建成功后自动选中新歌单：设置 `activePlaylistId` + `activeLibraryTab`
